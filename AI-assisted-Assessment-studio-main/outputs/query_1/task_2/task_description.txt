### Mythological Weight Balancer

In ancient times, weights representing values of different mythological creatures were used to balance scales accurately. This challenge revolves around calculating and verifying if a specified list of creature weights will balance the mythical scales.

**Task**:

Create a Python function `creature_balance(creature_weights_A, creature_weights_B)` that takes in two lists of positive integers:

- `creature_weights_A`: a list representing weights of creatures on side A of the scale.
- `creature_weights_B`: a list representing weights of creatures on side B of the scale.

The function should return a string:
- Return `'Balanced'` if the total weight on both sides A and B are equal.
- Return `'Side A'` if the total weight on side A is greater.
- Return `'Side B'` if the total weight on side B is greater.

You will add up all the weights of the creatures on each side and compare these totals to determine the scale's balance.

### Examples
```
creature_balance([5, 10, 5], [7, 13])
# returns 'Balanced'

creature_balance([10, 20], [5, 5])
# returns 'Side A'

creature_balance([5, 8], [10, 5])
# returns 'Side A'

creature_balance([12, 15], [25, 3])
# returns 'Side B'

creature_balance([7, 8], [9, 6])
# returns 'Balanced'
```

**Constraints**:
- Each list will contain between 1 and 10 integers.
- Each integer will be between 1 and 100 inclusive.
