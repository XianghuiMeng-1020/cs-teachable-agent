### Beginning of task description
In this task, you have to create a mini lottery simulation using a text file as the ticket storage. The program involves creating, managing and evaluating lottery tickets. 

Your task is to write a function `game_of_chance()` that accepts three arguments: `input_file` (name of the ticket storage file), `ticket_number` (a string representing a ticket number), and `draw_number` (a string representing a draw number). The function should:

1. Read the `input_file`, which contains a list of ticket numbers, one per line.
2. Check if the `ticket_number` is present in the file.
3. If the ticket exists, compare the length of both `ticket_number` and `draw_number`. Then:
   - If the length of `ticket_number` is greater than `draw_number`, return "JACKPOT".
   - If the length of `ticket_number` is equal to `draw_number`, return "WIN".
   - If the length of `ticket_number` is less than `draw_number`, return "NO WIN".
4. If the `ticket_number` does not exist in the file, return "TICKET NOT FOUND".

*Example:*
Assume the content of `lottery_tickets.txt` is:
```
1234
5678
91011
11213
```
- `game_of_chance('lottery_tickets.txt', '1234', '5678')` should return "WIN".
- `game_of_chance('lottery_tickets.txt', '12345', '123')` should return "JACKPOT".
- `game_of_chance('lottery_tickets.txt', '7890', '123456')` should return "TICKET NOT FOUND".

You can assume that the ticket numbers and draw numbers will be valid numeric strings and that the file exists with at least some tickets in it.
