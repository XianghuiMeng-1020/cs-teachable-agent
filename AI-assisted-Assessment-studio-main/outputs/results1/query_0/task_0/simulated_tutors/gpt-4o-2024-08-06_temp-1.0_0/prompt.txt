## Task: Lottery Number Evaluation

You are to create a Python program that simulates a simple lottery number evaluation system for a 'Game of Chance'. The game works as follows:

- Each player picks a 3-digit number (e.g., 579), where each digit is between 0 and 9 inclusive.
- A winning 3-digit number will be randomly drawn at the end of each week.

Your task is to write a function `evaluate_lottery_tickets(input_file, output_file, winning_number)` that evaluates a file containing multiple picked numbers, determines the number of winners, and writes them into an output file along with additional information.

### Constraints:
1. The function will read from `input_file`, a text file where each line contains a single 3-digit picked number as a string (e.g., '579'), and it contains no empty lines.
2. The `winning_number` is a string representing the winning 3-digit number (e.g., '632').
3. The output should be written into `output_file`, which includes:
   - The total number of tickets evaluated.
   - The number of winners.
   - Each winning ticket number on a new line, prefixed with 'Winner'.
   - If there are no winners, write 'No winners' instead.
4. You must handle only valid 3-digit numbers, no additional validations are required.
5. The function does not need to return any value.

### Example

Suppose `input_file` content is:
```
123
456
579
632
```

And the `winning_number` is '632'. The output file should look like:
```
Tickets evaluated: 4
Winners: 1
Winner: 632
```
