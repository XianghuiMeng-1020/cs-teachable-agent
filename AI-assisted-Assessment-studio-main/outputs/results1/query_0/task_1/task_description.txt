Create a Python program that simulates a simple dice game of chance. In this game, a player rolls two six-sided dice. The outcome of these dice rolls is recorded into a text file named 'game_log.txt'. The game should determine if the player wins, loses or ties based on the following rules:\n\n- If the sum of the dice is 7 or 11, the player wins.\n- If the sum is 2, 3, or 12, the player loses.\n- Any other sum results in a tie.\n\nThe game should: \n\n1. Take as input the two results of the dice rolls, which are integer values between 1 and 6 (inclusive).\n2. Calculate the sum of the dice results.\n3. Determine the outcome of the roll based on the sum using the rules provided.\n4. Write the result to the 'game_log.txt' file in the format: `dice1, dice2 - Outcome`. For example, a roll of 2 and 5 with a sum of 7 should be recorded as `2, 5 - Win`.\n\nImplement this functionality in a function `play_dice_game(dice1, dice2)`. This function does not return any value, but instead writes the result to the 'game_log.txt' file. The input values to the function will always be valid integers within the specified range.
