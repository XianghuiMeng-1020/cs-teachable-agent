Create a Python function `compute_total_calories(filename)` that reads a text file where each line contains the name of the dish and its associated calories separated by a comma. The function should return a string indicating the name of the dish with the highest calorie content and its respective calorie count in the format: 'DishName: Calories'. If the file is empty or if a line is not formatted correctly (i.e., doesn't contain exactly one comma), the function should return 'No valid data'.
