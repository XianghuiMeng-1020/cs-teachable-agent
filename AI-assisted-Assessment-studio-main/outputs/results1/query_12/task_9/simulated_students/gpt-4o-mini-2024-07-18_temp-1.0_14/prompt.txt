### Beginning of task description
Create a program in Python to analyze the calorie content of recipes stored in a text file. Each line in the file contains the name of a recipe, followed by the number of calories it contains. Your program should read from a file named `recipe_calories.txt` and create a single function `find_low_cal_recipes(threshold)`. This function should:

1. Accept a single integer `threshold` as an argument.
2. Read the `recipe_calories.txt` file.
3. Identify and return a list of recipe names whose calorie count is less than or equal to the provided `threshold`.

The program should handle the following scenarios:
- If the file is empty, the function should return an empty list.
- Ensure that the file can have any number of recipes, with one recipe per line.
- If no recipes are under the threshold, return an empty list.

Remember, each line in `recipe_calories.txt` is structured as:
```
recipe_name: calories
```
where `recipe_name` is the name of the recipe and `calories` is an integer value representing its calorie content.
