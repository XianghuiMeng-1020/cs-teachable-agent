### Beginning of task description
You are asked to write a Python function named `calculate_ingredient_costs` to help with managing cooking ingredient costs in a recipe. This function will read ingredient costs from a text file and calculate the total cost of required ingredients for a dish. The file will be formatted such that each line contains an ingredient name and its cost separated by a colon (e.g., `sugar:1.50`).

The function signature should be:

`calculate_ingredient_costs(filename, quantities)`

- `filename`: a string representing the name of the text file that contains the ingredient prices.
- `quantities`: a dictionary where keys are ingredient names (strings) and values are the amount needed (integers or floats) of each ingredient.

The function should return the total cost of the ingredients, calculated by multiplying the cost from the file by the quantity specified in the `quantities` dictionary. If an ingredient in the quantities dictionary does not appear in the file, it should be ignored. In case the file cannot be opened, the function should return `None` without raising an exception.

Example:
If the file `ingredients.txt` contains:
```
sugar:1.50
flour:0.75
eggs:0.50
```
And `quantities` is `{'sugar': 2, 'flour': 5, 'eggs': 1}`, the function would return `9.25`.
