Create a function `calculate_dish_time(dish_times)` that calculates the total cooking time for a series of dishes. Each entry in the input list `dish_times` represents the time required for a dish in minutes and can either be an integer or a string. If a string cannot be converted to an integer, it should be ignored. If a list item is missing or invalid, it should also be ignored. The function should return the total cooking time for all valid entries in the list.

For example:
- `calculate_dish_time(['60', 90, 'abc', '30'])` should return `180` (60 + 90 + 30) ignoring 'abc'.

The function will be tested with various lists including:
1. Different combinations of valid integers and strings.
2. Completely invalid entries.
3. A mix of valid and invalid entries.
4. Empty lists.
