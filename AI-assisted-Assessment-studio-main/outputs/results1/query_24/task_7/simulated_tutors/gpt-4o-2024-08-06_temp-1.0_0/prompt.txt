### Beginning of task description
You are to create a Python function to simulate a dice roll in a board game and calculate the cumulative score from multiple dice rolls. Implement a function `calculate_score(moves)` where the parameter `moves` is a list of strings representing dice rolls. Each string can be either a valid integer between '1' and '6' or the word 'skip' which indicates the player's turn was skipped and no dice was rolled.

Your task is to:
1. Compute the cumulative score by summing up the valid dice rolls. If a move is 'skip', it should be ignored.
2. If any move contains invalid data (neither a number between '1' and '6', nor 'skip'), skip that move and continue processing the rest.
3. Return the final cumulative score as an integer.

Notes:
- Do not assume the input list is restricted to a certain number of moves.
- Handle exceptions that may arise from invalid inputs gracefully.

Examples:
- `calculate_score(['1', '3', 'skip', '5', '4'])` should return `13`.
- `calculate_score(['2', 'skip', 'skip', '6', '4'])` should return `12`.
