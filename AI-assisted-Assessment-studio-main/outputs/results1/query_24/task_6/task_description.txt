Create a Python function `roll_dice(input_str)` that simulates rolling dice in a board game. The function takes a single string input `input_str` containing one or more "rolls" separated by commas. Each "roll" is a string in the format 'Xdy', where X is the number of dice to roll (an integer), and y is the number of sides on each die (an integer). The function should return a list where each element is the sum of the results from one "roll". The sum for a roll is the total of rolling X y-sided dice once. 

For instance, an input string '2d6,3d4' means rolling two 6-sided dice and three 4-sided dice. The function should handle exceptional cases where the format is incorrect or contains invalid numbers by returning an error message 'Invalid roll format' or 'Invalid number of dice or sides'. If `X` or `y` is not a valid integer or if either is less than 1, it should be considered invalid.

Do not use random number generation; instead, simulate the dice rolls by summing the maximum possible roll value (X * y) to give a deterministic outcome.
