You are tasked with creating a function called `board_game_summary` that analyzes a string representing the results of a multiplayer board game. In this game, the players are identified by single uppercase letters ('A' to 'Z'), and each player's score is represented by positive integers. The results string is formatted as pairs of character and number, indicating the player's award received multiple times during the game.

For example `"A12B4A3B10C1"` translates to Player 'A' getting two awards of 12 and 3 points, Player 'B' getting awards of 4 and 10 points, and Player 'C' getting an award of 1 point.

Your task is to implement the `board_game_summary(results)` function, where `results` is a `string`, and the function returns a dictionary where each key is a player and the corresponding value is the player's total score (sum of all awards received by that player).

The function should handle invalid data gracefully. If the results string is improperly formatted or if a player's score cannot be determined, the function should raise a `ValueError` with a descriptive message indicating the issue.
