The `<winner>` is a single uppercase letter ('A' to 'Z') representing the winning player, and `<score1>` and `<score2>` are integer scores of the two players involved. The function should calculate the total score for the winning player by adding the difference `(score1 - score2)` to its current total score. If there is an invalid entry format in game_results, the function should ignore that entry without throwing an error.

The function must return a dictionary where keys are player identifiers, and values are their total scores.
