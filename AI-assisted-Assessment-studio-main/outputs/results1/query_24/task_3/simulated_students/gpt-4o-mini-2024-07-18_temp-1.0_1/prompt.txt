### Beginning of task description
Create a function `calculate_board_game_score(game_results)` that calculates total scores for a series of board game matches based on a given score list. The input `game_results` is a list of strings where each string represents the results of a game in the format `"<winner>:<score1>,<score2>"`. The `<winner>` is a single uppercase letter ('A' to 'Z') representing the winning player, and `<score1>` and `<score2>` are integer scores of the two players involved. The function should calculate the total score for the winning player by adding the difference `(score1 - score2)` to its current total score. If there is an invalid entry format in game_results, the function should ignore that entry without throwing an error.

The function must return a dictionary where keys are player identifiers, and values are their total scores.
