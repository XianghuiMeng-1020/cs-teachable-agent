### Beginning of task description
Create a Python function `calculate_winner_score(filepath)` that takes a file path as a string argument. The file at the given path contains a summary of a board game scored by various players. The file is expected to have lines of format `player_name score`, where `player_name` is a string representing the player's name and `score` is an integer representing their score.

Your task is to read from the file and determine the player with the highest total score. Note that players may appear multiple times in the file. You should return a tuple of the player's name and their total score as an integer (name, total_score). If multiple players have the highest score, return any of them. Handle exceptions where the file might not be found or contain invalid data.

Example file content:
```
Alice 20
Bob 35
Alice 30
Charlie 25
Bob 40
```
For the above file, the function should return ('Bob', 75) since Bob has the highest cumulative score (35 + 40).
