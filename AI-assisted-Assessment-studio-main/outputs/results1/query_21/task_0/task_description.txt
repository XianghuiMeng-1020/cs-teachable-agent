Design a Python program to manage scores for a board game tournament. You need to implement a class `BoardGameTournament` to record and compute participants' scores. 

### Class Definition

**Class**: `BoardGameTournament`

- **Attributes**:
  - `participants`: a dictionary where keys are participant names (strings) and values are scores (integers).

- **Methods**:
  1. `add_participant(name)`: Adds a participant with the given name to the tournament with an initial score of 0.
  2. `record_score(name, score)`: Updates the score for the specified participant. If the participant doesn't exist, raise a `ValueError`.
  3. `get_score(name)`: Returns the current score of the specified participant. If the participant doesn’t exist, raise a `ValueError`.
  4. `save_scores(filename)`: Saves all participants and their scores to a file. Each line in the file should have format `participant: score`.
  5. `load_scores(filename)`: Loads participants and scores from a file. It should replace the current participants with those in the file. The file is assumed to have lines in the format `participant: score`.

### Task

Implement the `BoardGameTournament` class and all its methods. The scores are stored as integers and can be negative. The filename in `save_scores` and `load_scores` is a string.
