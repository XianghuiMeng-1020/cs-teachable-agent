### Task Description

In ancient times, druids and sages stored knowledge of mythical creatures in sacred texts. Your task is to create a program that processes such a text file containing information about mythical creatures and outputs statistics based on them.

You are provided with a text file named `creatures.txt`. Each line in the file describes a mythical creature with its name followed by a dash (`-`) and its category. Categories may include 'Dragon', 'Phoenix', 'Griffin', etc. For example:

```
Fire Drake - Dragon
Ice Phoenix - Phoenix
Storm Griffin - Griffin
...
```

Write a function `count_creatures(input_file, output_file)` that reads the `creatures.txt`, counts the number of creatures in each category, and writes these statistics to `output_file`. The output format should be one category per line, with the category name followed by a colon and a space, then the count. For example:

```
Dragon: 10
Phoenix: 5
Griffin: 7
...
```

### Testing

- The function will be tested using Pytest.
- Note that creature names are not unique and may appear under the same category multiple times.
- The order of categories in the output doesn't matter.
- Case sensitivity matters. Treat different cases as distinct.
- Assume the input file is correctly formatted.

### Examples


Your `count_creatures` function should output to another file, e.g., `output.txt`, that appropriately summarizes the categories and their counts.
