**Planetary Weather Log Analyzer**

In the year 2450, Earth has established Weather Research Stations on various planets in different star systems. Each research station logs weather data of its respective planet daily. Your task is to write a Python program that analyzes this weather data.

The weather data for each station is stored in individual text files. Each file has a name format of `StationX.txt`, where `X` is a unique number identifying the station. The contents of each file consist of daily average temperatures (in Celsius) recorded sequentially over several days, with each temperature stored on a separate line. All temperatures are represented as floating point numbers.

You need to implement a function `analyze_weather(station_id)`, where `station_id` is an integer representing the unique station number. This function should read the respective `.txt` file (e.g., `Station23.txt` for `station_id = 23`), compute the average temperature, and return it.
