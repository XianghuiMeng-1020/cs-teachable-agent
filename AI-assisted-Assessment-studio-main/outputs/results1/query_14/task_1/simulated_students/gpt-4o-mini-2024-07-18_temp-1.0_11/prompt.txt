You are tasked with creating a Python function that manages a cookbook which records recipes and their ingredients.    - 'update': Updates the ingredients of an existing recipe.
   - 'delete': Removes a recipe from the cookbook.
3. `recipe`: A tuple containing the recipe name and a list of its ingredients. For 'add' and 'update' operations, this will include the recipe name and a list of ingredients. For 'delete', this will include only the recipe name and an empty list.

The function should return the updated cookbook after performing the specified operation.
