Create a function `recipe_calculator` that takes two arguments: a recipe dictionary and a multiplier as an integer. The recipe dictionary contains ingredient names as keys and their quantities as string values (e.g., "2 cups of flour"). The function should return a new dictionary, where each ingredient's quantity is multiplied by the multiplier and rounded to the nearest whole number if necessary. The unit of each ingredient should remain unchanged in the string.

Function signature: `recipe_calculator(recipe, multiplier)`

Consider that the quantities are always integers in the initial dictionary, and will only need to be rounded when multiplied. Assume all quantities and the multiplier are positive and greater than zero.

The program will be tested using the `pytest` framework for validating the function's correctness with multiple scenarios.
