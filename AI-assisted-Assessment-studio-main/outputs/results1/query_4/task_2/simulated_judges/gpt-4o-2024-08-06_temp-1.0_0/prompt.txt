### Task Description

Create a Python function `play_dice_game(target)` that simulates a simple dice game where players take turns rolling a six-sided die. The function should return the number of rolls taken to reach a target score. The game follows these rules:

1. Players start with a score of 0.
2. Players take turns rolling a six-sided die, where each roll adds a random number between 1 and 6 to the player's score.
3. The game continues until a player's score meets or exceeds the target score specified by the input parameter.
4. The function should return the total number of die rolls taken to reach or exceed the target score.

The task will not involve randomness in the actual solution, as we will determine the number of rolls logically rather than through simulation. You can assume the target score is always a positive integer.

### Examples:

- `play_dice_game(5)` might return `1` or `2` depending on the rolls.
- `play_dice_game(10)` will calculate the minimal number of rolls required logically to meet or exceed the score of 10.
