### Beginning of task description
Design a Python function `play_roulette(starting_amount, rounds, bets)` simulating a simple game of chance - Roulette. In this game variant, the player starts with a certain amount of money and bets on a sequence of predefined rounds. The player can bet on 'red' or 'black'. Each color has an equal probability of winning, and each win doubles the player's bet amount, while a loss means the loss of the bet amount from the available funds. If the player's funds drop to zero before finishing the specified number of rounds, the game ends prematurely.

The function should take:
1. `starting_amount` (integer) representing the initial amount of money the player has.
2. `rounds` (integer) specifying the number of rounds to play.
3. `bets` (list of tuples) where each tuple contains a string ('red' or 'black') and an integer representing the amount of money the player bets on that color for each round.

Return a dictionary with the following keys:
- `'remaining_amount'` indicating the final amount of money the player has after the game.
- `'played_rounds'` indicating the number of rounds that were actually played before funds were exhausted or all betting rounds were completed.

Hints:
- You can assume the input values are valid and bets do not exceed the current funds available before starting any round.
- Use a random integer generator to simulate winning/losing outcomes of each round (0 for 'red' wins, 1 for 'black' wins).

Implement this function using only variables, arithmetic operators, selection statements, loops, and dictionaries.
