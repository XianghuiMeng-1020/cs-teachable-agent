Create a game simulation called 'Lucky Number'. The game involves rolling a 6-sided dice until a "lucky" number is rolled. A lucky number is any number between 1 and 6 that yields a positive net gain for the player. The player starts the game with 0 points, and for every roll, their score is determined by the value shown on the dice and the corresponding multiplier:

Here is how points are assigned based on dice roll:

- If the dice shows 1: 0 points
- If the dice shows 2: +10 points
- If the dice shows 3: -5 points
- If the dice shows 4: +5 points
- If the dice shows 5: +20 points
- If the dice shows 6: -10 points

The game should continue rolling the dice until the player's net gain is positive. This means if they start losing points after rolls, the game will continue rolling until the cumulative total score is positive.

Implement a function `lucky_number_game()` that runs the game simulation and returns the number of dice rolls taken to achieve a positive net gain for the player.

The score mapping should be implemented using a dictionary. Do not use input/output operations. The function should just return the number of rolls required to end the game with a positive score.
