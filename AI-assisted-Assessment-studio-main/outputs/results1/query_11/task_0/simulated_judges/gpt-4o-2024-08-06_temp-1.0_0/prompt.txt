### Task Description

You are a chef who needs to manage multiple recipes in your cookbook. Each recipe contains ingredients and the quantity required for a specific dish. You want to create a Python program to keep track of the ingredients and calculate total quantities required for any given recipe. Here is what you need to do:

1. Define a function `calculate_ingredients(recipe_list, recipe_name)` that accepts a list of multiple recipes and the name of the recipe you want to calculate.
   - Each recipe in the list is a tuple of the form: `("Recipe Name", [("Ingredient1", quantity1), ("Ingredient2", quantity2), ...])`.
   - The `quantity` in the tuple will represent the amount needed for that specific ingredient.
   
2. The function should return a list of tuples containing each ingredient needed and its total quantity for the specified `recipe_name`. If the recipe does not exist in the `recipe_list`, return an empty list.

### Example

```python
recipe_list = [ 
    ("Pasta", [("Flour", 100), ("Egg", 2), ("Salt", 1)]),
    ("Omelette", [("Egg", 3), ("Salt", 1), ("Oil", 1)]),
    ("Cake", [("Flour", 200), ("Egg", 3), ("Sugar", 100), ("Salt", 1)])
]

print(calculate_ingredients(recipe_list, "Cake"))  
# Output: [('Flour', 200), ('Egg', 3), ('Sugar', 100), ('Salt', 1)]
```

### Notes:
- You must iterate through the `recipe_list` and search for the specific recipe.
- Ensure that your solution handles cases where the requested recipe is not found by returning an empty list.
- Assume the `recipe_list` is not empty, but consider that the requested recipe might be absent.
