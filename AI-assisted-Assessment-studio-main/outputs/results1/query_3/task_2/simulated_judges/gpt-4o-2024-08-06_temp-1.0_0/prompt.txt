Create a Python function, `play_game_of_chance(strategy)`, where `strategy` is a dictionary containing betting outcomes and their respective bets in a game of chance. The function should simulate a simple roulette game where there are two main outcomes: 'win' and 'lose'.

The dictionary `strategy` contains keys 'win' and 'lose', which correspond to the amount of money bet on winning or losing, respectively.

The game rules are as follows:
- If a player wins: they receive twice the amount of their bet.
- If a player loses: they lose the amount of their bet.

The function should return a dictionary with keys 'result' and 'balance'. The value for 'result' should be a string indicating whether the player wins ('win') or loses ('lose'). The value for 'balance' should represent the net gain or loss of the player after the bet is resolved based on a 50% chance of winning.

To simulate the outcome of the roulette, assume an equal 50% chance for 'win' and 'lose'. Use the length of keys in `strategy` modulo 2 to determine the outcome (even for 'win', odd for 'lose'). Note that other than calculating the modulus, the actual implementation of random number generation or random module usage is not required.

Example:
```python
strategy = {'win': 100, 'lose': 50}
result = play_game_of_chance(strategy)
# The result could be {'result': 'win', 'balance': 100} or {'result': 'lose', 'balance': -50} depending on the calculated outcome.
```
