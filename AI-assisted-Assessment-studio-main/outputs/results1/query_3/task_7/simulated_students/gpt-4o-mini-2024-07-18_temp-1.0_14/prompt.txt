Create a Python function named `simulate_lottery` that mimics a simple lottery game. The lottery game involves selecting 6 numbers between 1 and 49. The function should take two parameters. Each key in this dictionary is a round number (as an integer), and each value is a list containing 6 unique integers between 1 and 49 inclusive, representing the winning numbers for that round.

The function's task is to determine the number of rounds in which the player has won by matching at least 4 numbers. The function should return the total number of rounds won by the player.

Ensure that your solution handles edge cases, such as:
- The player selecting numbers that do not match the winning numbers in any round.
- All player numbers matching winning numbers in a round.

Your solution should not assume the input to be validated and should handle incorrect number selections by returning zero in such cases (if any player or drawn numbers are out of the allowed range or invalid lengths).

Example:
```python
rounds = {
    1: [4, 15, 23, 29, 35, 40],
    2: [3, 19, 22, 34, 41, 47],
    3: [5, 12, 23, 29, 37, 42]
}
player_numbers = [5, 12, 23, 29, 37, 1]
simulate_lottery(player_numbers, rounds) # returns 2
```
