### Beginning of task description
In this programming task, you are required to create a simple 'Game of Chance'. Your goal is to simulate a basic dice roll game and calculate the total score for a series of games. Implement a function `play_dice_games(games)`, where `games` is a dictionary representing a series of games played. Each key in the dictionary is the player's name, and each value is a list of numbers representing the outcomes of the dice rolls in their respective game. Each dice roll can have an integer value from 1 to 6, inclusive.

Your function should compute and return a dictionary that maps each player's name to their total score. The score of a player is simply the sum of their dice roll outcomes. For example, if a player's dice rolls are `[3, 5, 1]`, their score would be `9`.

The output dictionary should list the players' scores in the same order as they appear in the input dictionary. Additionally, include a check to ensure a player has a roll of exactly 3 dice. If a player has rolled less or more than 3 dice, their score should be set to `-1` indicating an invalid game.

### Example
Input:
```python
{
  'Alice': [3, 5, 2],
  'Bob': [1, 6, 3],
  'Charlie': [4, 4, 4],
  'David': [2, 2]
}
```
Output:
```python
{
  'Alice': 10,
  'Bob': 10,
  'Charlie': 12,
  'David': -1
}
```
