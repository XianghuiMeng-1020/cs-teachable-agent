Create a function `calculate_payout(bets, winning_number)` that simulates a simple roulette game.

The function should take two arguments:
1. `bets`: a list of dictionaries where each dictionary represents a player's bet. Each dictionary contains:
   - `"player"`: the player's name (a string).
   - `"bet_type"`: the type of the bet (either `"number"`, `"even"`, or `"odd"`).
   - `"amount"`: the betting amount (an integer).

2. `winning_number`: an integer (0-36) representing the winning number from the roulette spin.

Your task is to calculate the payout for each player based on the following rules:
- If `bet_type` is `"number"` and matches `winning_number`, payout is 35 times the amount bet.
- If `bet_type` is `"even"` and `winning_number` is even (and not 0), payout is 2 times the amount bet.
- If `bet_type` is `"odd"` and `winning_number` is odd, payout is also 2 times the amount bet.
- Otherwise, the payout is 0.

The function should return a dictionary where each key is a player's name and each value is the total payout for that player.

Example:
```python
bets = [
    {"player": "Alice", "bet_type": "number", "amount": 10},
    {"player": "Bob", "bet_type": "even", "amount": 15},
    {"player": "Charlie", "bet_type": "odd", "amount": 5}
]
winning_number = 2
calculate_payout(bets, winning_number) 
# should return {"Alice": 0, "Bob": 30, "Charlie": 0}
```
