   - `get_score()`: Returns the current `highest_score` for the player.

3. **String Representation**:
   - Implement the `__str__` method, which returns a string containing the player's name and highest score in the format `"Player: <name>, Highest Score: <highest_score>"`.

Write a function `parse_scores(data)` that accepts a string `data` with scores in the format `"Name1:5,Name2:3,Name1:10,Name2:6"`. Use this function to create `PlayerScore` objects for each player and update their highest scores according to the scores in the input string.

Returns a list of `PlayerScore` objects representing each player and their highest score.
