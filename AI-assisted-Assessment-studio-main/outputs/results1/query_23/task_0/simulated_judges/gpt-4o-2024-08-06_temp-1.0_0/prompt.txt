You are tasked with creating a simple board game scoring system in Python. The board game involves multiple rounds, and each player keeps track of their cumulative score. The players' scores are stored in a list associated with each player. You need to implement a `BoardGame` class with the following requirements:

1. The class should have a constructor that initializes the game with an empty list of player's scores.
2. Implement a method `add_player(self, player_name)` that adds a player to the game with an initial score of an empty list.
3. Implement a method `record_score(self, player_name, score)` that appends the `score` to the list of scores for the specified `player_name`.
4. Implement a method `get_player_total_score(self, player_name)` that returns the total score across all rounds for the specified `player_name`.
5. Implement a method `get_winner(self)` that returns the name of the player with the highest total score. In the event of a tie, return the name of the winner sorted alphabetically.

You can assume that all player names are unique and are always added before their scores are recorded. Functions will only be created for existent players. You will create a game object, add players, record their scores for different rounds, compute total scores, and determine the current winner based on the provided player scores.
