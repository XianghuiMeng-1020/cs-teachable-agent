### Task
You are a scribe in a mystical library dedicated to mythology. Your task is to create a program that reads from a text file containing the legends of different mythological characters. Each legend is preceded by a character's name on its own line and followed by a line containing just the word "END".

Your program should parse this file to determine the character with the longest legend based on the number of words, and write this character's name along with their legend into a new output file called `longest_legend.txt`.

### Input
- A text file `mythology_legends.txt` with the following structure:
```
Achilles
His strength is unparalleled. Known as the hero of the Trojan War.
END
Hercules
This son of Zeus is famous for his incredible strength and for completing the Twelve Labors.
END
...
```

### Output
- A file `longest_legend.txt` containing the name of the character followed by their longest legend.

### Requirements
- The program must correctly identify and handle multiple characters and their respective legends.
- The legends must be compared based on the number of words.

### Function Signature
Implement the function `find_longest_legend(input_filepath: str, output_filepath: str):`

Use this function to process `mythology_legends.txt` and generate `longest_legend.txt`.
