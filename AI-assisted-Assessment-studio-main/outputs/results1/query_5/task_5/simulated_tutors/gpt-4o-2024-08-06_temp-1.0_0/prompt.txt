The first integer, 'power', indicates the creature's maximum power score, and the second integer, 'skill', represents the creature's skill level.

A creature is classified as 'Legendary' if its power is greater than or equal to 90 and its skill level is greater than or equal to 75.

Your program should:
- Read from the `input_file`, where each line contains 'CreatureName Power Skill'.
- Determine if each creature is 'Legendary'.
- Write only the names of the 'Legendary' creatures, each on a new line, into the `output_file`.

The function should not return anything, just read from the `input_file`, process the information, and write to the `output_file`. You can assume that the file is well-formed and that each creature's data is specified correctly.
