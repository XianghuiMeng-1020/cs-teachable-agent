Your task is to create a function `calculate_power(filename)` which reads a file (with a given `filename`) that contains a single line of comma-separated integers representing the changes in megawatts. Your function should calculate and return the final power level of the subsystem, starting from an initial power level of 1000 MW. The power level is updated by the sum of all changes specified in the file. For example, if the file contains the following line:

```
-50, +10, -20
```

The final power level will be `1000 - 50 + 10 - 20 = 940` MW.

The function should return the computed power level as an integer.
