    `mass1,distance,mass2`

Where:
- `mass1` and `mass2` are the masses of two planets in kilograms (floats).
- `distance` is the distance between the two planets in meters (float).

Your function must compute the gravitational force between each pair of planets using the formula:

    F = (G * mass1 * mass2) / (distance ** 2)

Where:
- `F` is the gravitational force in Newtons (float).
- `G` is the gravity constant, which is approximately 6.67430 * 10^-11 N*(m^2)/kg^2.

The function should write the computed gravitational forces to a file named `output_filepath`. Each line in the output file should correspond to the gravitational force of the planetary pair from the input file, formatted to two decimal places.
