### Task Description

In a distant future, scientists have developed a system to monitor the energy consumption of various units on a space station. You are tasked with writing a Python program to help with this monitoring system. The program will analyze energy consumption data and generate reports based on input files.

The input file, named `energy_data.txt`, will contain multiple lines, where each line represents the energy consumption of a unit measured in kilowatts per hour (kWh). Each line in the file will have two parts separated by a comma: the unit's ID and the energy consumed.

Your task is to:
1. Read the file `energy_data.txt`.
2. Calculate the total energy consumed by all units.
3. Calculate the average energy consumption per unit.
4. Write a report to a file named `energy_report.txt` containing two lines:
   - The first line should state `Total Energy: X kWh` where X is the total energy consumed.
   - The second line should state `Average Energy: Y kWh` where Y is the average energy consumed per unit, rounded to two decimal places.

Ensure your program handles the cases where the file is empty or all the units consume 0 kWh without errors. 

Write a function `generate_energy_report(input_file, output_file)` that takes the name of the input file and the name of the output file as arguments to accomplish this task.

### Example

Suppose `energy_data.txt` contains the following lines:
```
Unit1,10
Unit2,20
Unit3,15
```
The generated `energy_report.txt` should contain:
```
Total Energy: 45 kWh
Average Energy: 15.00 kWh
```
