In the distant future, intergalactic trade is thriving due to advanced space travel capabilities. As a software engineer aboard the trading spaceship 'Argonaut', you are tasked with calculating the total fuel requirements for various trade missions.

Write a Python function `calculate_fuel_for_mission(filename)` which reads trade mission data from a file specified by `filename`. Each line in the file represents a mission and contains the distance to the destination planet (in light years) as an integer.

The fuel required for a mission is determined by the formula:
- fuel = distance * 2 (since 2 units of fuel are required per light year traveled)

Your task is to compute the total fuel required for all listed missions in the file. The total fuel should be written to a new file named `total_fuel.txt`. This file will contain a single integer - the total fuel required for all missions.

You are not required to validate the contents of the file. Assume the distance values provided are valid non-negative integers.

Example:
If the file 'missions.txt' contains:
```
10
20
15
```
Then 'total_fuel.txt' should contain:
```
90
``` 

Your solution will be tested using various test files with different sets of missions.
