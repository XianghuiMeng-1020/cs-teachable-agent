2. **Add a Recipe**: Implement a method `add_recipe(self, recipe_name)` that adds a new recipe to the list and writes it to `recipes.txt` if it doesn't already exist in the list (case sensitive check).

3. **Remove a Recipe**: Implement a method `remove_recipe(self, recipe_name)` that removes a recipe from the list and updates `recipes.txt`. If the recipe does not exist, it should handle the situation gracefully without any changes to the file.

4. **List Recipes**: Implement a method `list_recipes(self)` that returns the list of recipes.

Your implementation should handle any exceptions that might occur during file operations.

    manager = RecipeManager()
    manager.remove_recipe('Salad')
    assert sorted(manager.list_recipes()) == ['Omelette', 'Pancakes']
