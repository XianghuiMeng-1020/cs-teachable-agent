### Beginning of task description
You are asked to build a simple inventory management system for a kitchen. You will implement a class called `KitchenInventory` that allows for tracking availability of different ingredients. The functionality includes loading data from a file, checking ingredient availability, adding new ingredients, and saving back to the file.

**Specifications:**

1. Implement a class `KitchenInventory` with the following methods:
   - `__init__(self, filename)`: Initialize with the filename of the inventory.
   - `load_inventory(self)`: Load the inventory from a text file where each line contains 'ingredient_name:quantity' (e.g., 'sugar:5'). If the file does not exist, start with an empty inventory and handle exceptions appropriately.
   - `check_availability(self, ingredient)`: Check if a given ingredient is available in the inventory. Return True if available, otherwise False.
   - `add_ingredient(self, ingredient, quantity)`: Add a specified quantity of an ingredient to the inventory. If the ingredient already exists, increase its quantity. Otherwise, add it as new.
   - `save_inventory(self)`: Save the current inventory back to the file specified during initialization. Each ingredient and its quantity should be saved on a new line in the format 'ingredient_name:quantity'.

**Testing:**
