### Beginning of task description
You need to write a Python program for a fictional board game tournament where each game result is recorded in a file. Each line in the file contains the names of two players separated by a space, followed by the result of that game: either 'win' for the first player or 'lose' for the first player (which implies a win for the second player).

Your task is to define a function `calculate_scores(filename)` that reads the game results from this file, calculates the total number of wins for each player, and returns a list of players who have won at least one game, sorted by the number of wins in descending order. If two players have the same number of wins, their names should be sorted alphabetically.

Function signature: `def calculate_scores(filename):`

Here's how the program will be tested:
1. A file named `tournament.txt` will be provided with multiple lines representing game results.
2. The `calculate_scores` function will be called with `'tournament.txt'` as the argument.
3. Your function should return a list of tuples. Each tuple should contain a player's name and their win count.

Here is the format of the file content:
```
Alice Bob win
Charlie Alice lose
Bob Charlie win
```

In this example, the function should return `[('Alice', 2), ('Bob', 1), ('Charlie', 0)]`. Players with zero wins should be excluded from the result.
