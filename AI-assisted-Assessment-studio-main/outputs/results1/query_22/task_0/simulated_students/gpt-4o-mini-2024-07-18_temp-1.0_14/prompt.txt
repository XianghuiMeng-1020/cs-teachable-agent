Function signature: `def calculate_scores(filename):`

```
Alice Bob win
Charlie Alice lose
Bob Charlie win
```

In this example, the function should return `[('Alice', 2), ('Bob', 1), ('Charlie', 0)]`. Players with zero wins should be excluded from the result.
