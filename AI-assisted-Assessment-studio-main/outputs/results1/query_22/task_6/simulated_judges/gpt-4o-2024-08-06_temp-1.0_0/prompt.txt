### Task:
You are tasked to create a Python program that manages scores during a board game session.

Write a program with the following functions:

1. `log_scores(filename, players, scores)`: This function takes a filename (string), a list of players' names, and a matching list of scores. The function should write each player's name and their respective score to the specified file, in the format `name: score` on separate lines.

2. `top_score(filename)`: This function reads from the file with the name `filename` and returns the name(s) of the player(s) with the highest score as a list.

The format of the file will be as follows:
- Each line contains a player's name followed by their score, separated by a colon and a space (e.g., `Alice: 10`).
- There will be no duplicate player names.

### Example:
Suppose you call `log_scores('game_scores.txt', ['Alice', 'Bob', 'Charlie'], [10, 15, 20])`. The file 'game_scores.txt' will contain:
```
Alice: 10
Bob: 15
Charlie: 20
```
Calling `top_score('game_scores.txt')` will return `['Charlie']` as Charlie has the highest score.

### Constraints:
- The `players` and `scores` lists will always have the same length.
- If there are multiple players with the highest score, your function should return all their names in the same order they appear in the file.
- All scores are non-negative integers.
