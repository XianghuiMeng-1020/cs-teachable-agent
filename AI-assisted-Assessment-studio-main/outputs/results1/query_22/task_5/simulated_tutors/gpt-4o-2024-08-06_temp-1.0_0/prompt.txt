## Task Description

Create a Python function `calculate_winner_board_game(file_path)` that simulates reading a file containing board game score records and determines the winner of the game. Each line in the file represents a different player, their score, and a certain number of rounds they played. Each line in the file is in the format: `player_name:score:rounds`.

Your function should read the file, process each player's score and rounds, and determine which player has the highest `average score per round`. If two or more players have the same average score per round, the tie is broken by the player who played fewer rounds. You can assume that `rounds` will always be greater than 0.

Your program will be tested using different files as input, and you should ensure your function returns the name of the winning player as a string.

### Example format of input file:
```
Alice:150:3
Bob:120:4
Charlie:180:2
```

### Example output:
For the above file, `calculate_winner_board_game()` should return `'Charlie'` because Charlie has an average score of `90` per round which is higher than Alice's (`50`) and Bob's (`30`).

### Constraints:
- You can assume the input file is well-formed and follows the specified format.
- There are at least two players in the file.
