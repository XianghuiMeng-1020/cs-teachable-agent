Create a Python class `MythArchive` that manages a collection of mythological tales stored as text files. Each tale is represented by a file that contains its name on the first line and the tale's content on subsequent lines. Implement the following methods:

2. `get_tale(name)` - Returns the content of the tale with the given name as a string. If the tale is not found, return an empty string.

3. `remove_tale(name)` - Removes the tale with the given name from the archive. If the tale is not found, it should have no effect.

4. `list_tales()` - Returns a list of all tale names currently in the archive.
