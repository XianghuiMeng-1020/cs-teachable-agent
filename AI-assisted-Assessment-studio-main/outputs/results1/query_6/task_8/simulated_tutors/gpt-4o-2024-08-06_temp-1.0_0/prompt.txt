In the realm of mythology, there are countless heroic tales, mysterious creatures, and powerful gods. 

Your task is to create a Python program that reads from a text file named `mythology_stories.txt`. This file contains different mythology stories in a specific format:
```
# Story Title: Zeus and the Thunderbolt
Zeus, the king of the Greek gods, was known for many mythological exploits...
# Story Title: The Odyssey
Odysseus, the hero of The Odyssey, encountered many challenges...
```

Each story begins with a line that starts with `# Story Title:` followed by the title of the story. The subsequent lines contain the content of the story.

Implement a class `Mythology` with the following specifications:
- An instance variable `stories` that stores the stories in a dictionary where keys are story titles, and values are the story content.
- A method `read_stories_from_file(filename)` that populates the `stories` dictionary by reading the given text file.
- A method `get_story(titles)` that accepts a list of story titles and returns a list of story contents corresponding to the given titles.
- A method `save_story_to_file(title, filename)` that writes the content of the story with the given title into a new file specified by `filename`.

The method `read_stories_from_file` should be called automatically when an instance of the class is created.
