Welcome to the future where humans coexist with artificial intelligences (AIs) on distant planets. As a staff member on a space research vessel, you have been tasked with monitoring communication logs to detect automatic warning messages generated by AIs when the systems detect abnormalities. These logs are strings that sometimes contain the word "DANGER" when a warning is being issued.

Write a function `count_danger_messages(logs)` that takes a list of strings `logs`. Each string in the list represents a communication log from an AI system. Your task is to count the total number of times the word "DANGER" (case-sensitive) appears in the logs and return this count.

Function signature: `def count_danger_messages(logs):`

The input list `logs` can have a varying number of elements. Each string may contain multiple occurrences of the word "DANGER". The function must return an integer representing the total number of occurrences of the word "DANGER" across all logs.

Consider edge cases such as empty logs or logs without any "DANGER" messages.
