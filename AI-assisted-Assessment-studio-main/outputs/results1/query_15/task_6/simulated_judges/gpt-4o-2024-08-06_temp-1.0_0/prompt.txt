### Task: Alien Message Decoder

You have intercepted a message from an alien culture. The message is encoded as a string of lowercase characters in a fictional alien alphabet. The encoding rules dictate that each character in the message refers to a position in the standard English alphabet, where 'a' corresponds to 1, 'b' corresponds to 2, ..., 'z' corresponds to 26.

Write a function `decode_alien_message(message)` that takes a single `message` string as input and returns a list of integers representing the decoded values based on the above rules.

Your task is to implement the `decode_alien_message` function such that:
- It processes each character of the `message` string.
- Converts each character to its corresponding integer value.
- Returns a list containing these integer values.

Assumptions:
- The input `message` will only contain lowercase alphabetic characters, with a length of at least 1 and no more than 100.

**Examples:**

- `decode_alien_message('abc')` should return `[1, 2, 3]`.
- `decode_alien_message('zzz')` should return `[26, 26, 26]`.
- `decode_alien_message('hello')` should return `[8, 5, 12, 12, 15]`.

Your solution will be tested against several cases to ensure its correctness.
