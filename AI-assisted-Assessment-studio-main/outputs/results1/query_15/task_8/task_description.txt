Imagine a group of astronauts on an interplanetary exploration mission. They have a robotic assistant on board their spaceship that helps them communicate by encoding and decoding messages for secure transmission. The robot uses a basic encoding technique called 'vowel shift'. 

Each vowel in the English alphabet (a, e, i, o, u) is shifted to the next vowel in the sequence (a->e, e->i, i->o, o->u, u->a), and consonants remain unchanged. Your task is to implement a function named `encode_message(message)` that takes a string, `message`, as input and returns a new string where each vowel is replaced according to this shift.

The message will consist only of lowercase alphabetical characters and spaces. Spaces should be preserved in their original positions.

The function should follow the signature:

```
encode_message(message)
```

Your function will be tested with multiple test cases and should handle words with varying lengths, as well as messages containing only vowels, only consonants, or a mixture of both.
