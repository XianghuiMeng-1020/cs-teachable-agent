You are tasked with writing a Python function to decide board game outcomes based on dice roll values. The game outcome is determined by comparing the total value of rolls from two players. Implement the function `find_winner(player1, player2)` where `player1` and `player2` are strings representing the dice outcomes for each player.

Each player rolls two standard 6-sided dice, and their outcomes are given as strings in the format 'X Y', where X and Y are integers representing the numbers rolled on each die, for example '3 5'.

The function should calculate the total score for each player by adding their respective dice values. The function should return the string:
- "Player 1 Wins" if player 1's total score is greater than player 2's total score.
- "Player 2 Wins" if player 2's total score is greater than player 1's total score.
- "Draw" if both players have the same total score.

**Examples:**
- `find_winner('3 5', '4 2')` should return `'Player 1 Wins'`.
- `find_winner('6 3', '5 4')` should return `'Draw'`.

Write your implementation using basic operations like arithmetic computation and simple conditional statements.
