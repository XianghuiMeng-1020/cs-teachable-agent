In a distant future, humans have developed a fleet of spaceships for intergalactic travel. You are tasked with creating a Python class `FleetManager` that manages a collection of spaceships, each having a name, a maximum speed, and a list of missions it has undertaken.

Your implementation should include the following:

- Implement a class `Ship` with attributes:
  - `name`: a string representing the ship's name.
  - `max_speed`: an integer representing the maximum speed of the ship.
  - `missions`: a list of strings representing the missions the ship has undertaken.

- Implement a class `FleetManager` that manages a collection of `Ship` objects. It should have the following methods:
  
  - `add_ship(name, max_speed)`: Adds a new `Ship` with the specified name and max_speed to the fleet. If a ship with the same name already exists, it should raise a `ValueError`.
  
  - `record_mission(ship_name, mission)`: Records a new mission for the specified ship. If the ship does not exist in the fleet, it should raise a `KeyError`.
  
  - `get_missions(ship_name)`: Returns the list of missions for the specified ship. If the ship does not exist, it should return an empty list.
  
  - `get_fleet_summary()`: Returns a dictionary where keys are the names of the ships and values are dictionaries with two keys: `max_speed` and `missions`.
  
Your program will be tested using various scenarios to ensure that the fleet manager can handle querying and updating ship information as required.
