- `kingdom`: A string indicating the realm or domain where the creature is ruler or prominently dwells.

The function should return another dictionary which contains the following information:

- `total_creatures`: The total number of creatures.
- `average_power`: The average power of all creatures, rounded down to the nearest integer.
- `kingdom_distribution`: A dictionary that maps each unique kingdom to the number of creatures belonging to that kingdom.
