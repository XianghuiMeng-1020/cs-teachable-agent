### Task Description

You are required to create a Python program relevant to the theme 'Game of Chance'. The task revolves around a simple file-based betting game that involves analyzing the outcomes of dice rolls. Your task is to develop a function and handle the input/output through files.

#### Task Details:

1. You need to write a function `analyse_dice_game(input_file, output_file)` that takes two file paths as input:
   - `input_file`: A text file where each line represents a dice roll outcome of a game round, which is either "win", "lose", or "draw".
   - `output_file`: This file should be used to store the analyzed result of the game.

2. The function should read from the `input_file`, calculate the total number of 'win', 'lose', and 'draw' outcomes, and write these counts to the `output_file` in the following format:

```
Wins: <count_of_wins>
Loses: <count_of_loses>
Draws: <count_of_draws>
```

3. Ensure the output file ends with a newline character.

4. Assume that each line of the `input_file` will contain only "win", "lose", or "draw".

```

```

#### Output file (`output.txt`) after running your function:
