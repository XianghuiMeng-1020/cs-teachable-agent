### Task Description

The Galactic Federation requires your programming prowess to help them decipher alien messages. You have intercepted encoded messages sent from different planets in the Andromeda Galaxy to Earth. Each intercepted message is a string containing words that correspond to specific planetary codes.

Your task is to create a function `decode_message(galactic_message, codebook)` that takes in:
- `galactic_message`: a string representing an encoded message. The words in the message (separated by spaces) are keys you can use to find their corresponding decoded words.
- `codebook`: a dictionary where each key is a code from a planetary language and its value is the decoded meaning in English.

Your function should return the decoded message. If a word in `galactic_message` does not exist in the `codebook`, it should be replaced with 'UNKNOWN'. All words should be joined together by a single space in the output, and there should be no trailing spaces at the beginning or end.

### Example

```python
codebook = {'QXZ':'hello', 'BFO':'world', 'DZT':'alien'}
galactic_message = 'QXZ BFO GAO'

decode_message(galactic_message, codebook)
# Output: 'hello world UNKNOWN'
```

Your solution will be tested against several encoded messages to ensure its correctness. Be sure to handle cases where there are no recognized words.
