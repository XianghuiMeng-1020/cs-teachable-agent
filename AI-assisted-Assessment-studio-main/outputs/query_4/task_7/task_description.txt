### Task: Board Game Victory Tracker
In this task, you will implement a `track_wins` function to track the number of victories across multiple board games. The program should handle various board games and keep track of victories for individual players consistently.

### Function Signature:
`def track_wins(games):`
- `games`: List of tuples. Each tuple represents a game instance in the format `(game_name, winner_name)`. The list may contain the same game but played at different times, which would be represented by multiple tuples.

### Requirements:
1. The function should return a dictionary where each key is the player's name and the value is another dictionary.
2. The inner dictionary should have the game names as keys and the number of wins as values.
3. If a player has not won any game, their name should not appear in the dictionary.
4. Consider edge cases like an empty list or cases where certain players have never won any games.

### Example:
```python
track_wins([('chess', 'Alice'), ('checkers', 'Bob'), ('chess', 'Alice'), ('monopoly', 'Bob'), ('checkers', 'Alice')])
```
**Output:**
```python
{
  'Alice': {
    'chess': 2,
    'checkers': 1
  },
  'Bob': {
    'checkers': 1,
    'monopoly': 1
  }
}
```
