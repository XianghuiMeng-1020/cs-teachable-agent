You need to implement a function `update_leaderboard(rounds)` that accepts a list of dictionaries, each representing a game and containing player names as keys and their respective points as values for that round.

The program should maintain a cumulative leaderboard where each player's total score accumulates across all rounds. The leaderboard should be stored as a dictionary, with player names as keys and their total scores as values. The function should return the final leaderboard sorted in descending order based on total scores.
