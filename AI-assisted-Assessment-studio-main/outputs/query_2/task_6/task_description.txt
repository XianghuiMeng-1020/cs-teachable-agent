### Task Description

Write a function `generate_shopping_list(recipes_file, available_file, output_file)` that reads a file of recipes and a file of available ingredients, compares the two, and generates a shopping list of missing ingredients to an output file.

**Detailed Instructions:**
1. The function will take three parameters:
    - `recipes_file`: the name of a text file containing a list of recipes. Each recipe starts with a line with the recipe name followed by the ingredients, one per line.
    - `available_file`: the name of a text file containing a list of available ingredients, one per line.
    - `output_file`: the name of the output file where the function will write the shopping list.
2. Ingredients will be compared in a case-insensitive manner.
3. Each recipe in the `recipes_file` will be separated by an empty line.
4. Your function should write each missing ingredient to the `output_file`, with one ingredient per line.

### Example
#### recipes_file content:
```
Pasta
Tomato
Basil
Garlic
 
Salad
Lettuce
Tomato
Cucumber
```
#### available_file content:
```
Tomato
Lettuce
Salt
Oil
```
#### After running `generate_shopping_list('recipes.txt', 'available.txt', 'shopping_list.txt')`, the `shopping_list.txt` should contain:
```
Basil
Garlic
Cucumber
```
