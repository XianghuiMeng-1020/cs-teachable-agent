## Task Description

You are tasked with developing a Python program that will help manage a collection of cooking recipes stored in a file. Each recipe consists of a title, a list of ingredients, and cooking instructions. These recipes should be manipulative as a list of strings named `recipes.txt`. Each recipe in the file is formatted as follows:

```
Recipe Name
Ingredient 1, Ingredient 2, ...
Instruction Line 1
Instruction Line 2
...
```

Your program should include a function called `organize_recipes(filename)`, which extracts and organizes the recipes from the file into a list of dictionaries. Each dictionary should contain the keys: `title` (the recipe's title), `ingredients` (a list of ingredients), and `instructions` (a list of instruction lines).

The recipe list should be returned sorted alphabetically by the recipe title.

### Example

Given a file `recipes.txt` with the following content:

```
Pasta
Tomato, Basil, Garlic, Pasta
Boil pasta
Cook sauce
Mix together

Cake
Flour, Sugar, Eggs, Butter
Mix dry ingredients
Add eggs and butter
Bake
```

Calling `organize_recipes('recipes.txt')` should return:

```
[
    {'title': 'Cake', 'ingredients': ['Flour', 'Sugar', 'Eggs', 'Butter'], 'instructions': ['Mix dry ingredients', 'Add eggs and butter', 'Bake']},
    {'title': 'Pasta', 'ingredients': ['Tomato', 'Basil', 'Garlic', 'Pasta'], 'instructions': ['Boil pasta', 'Cook sauce', 'Mix together']}
]
```
