## Task Description: 

Welcome to the Lucky Roll Game! Your task is to simulate a simple game of chance using Python. The game involves rolling a 6-sided die multiple times. Each roll results in a score that contributes to the final result. However, there's a twist: special rules determine the value added to your score based on the roll result.

### Rules:
1. Roll a 6-sided die. Each die has sides numbered from 1 to 6.
2. If the roll result is 6, this results in a score of 0 for that roll.
3. If the roll result is an even number (but not 6), the score for that roll is twice its number.
4. If the roll result is an odd number, the score for the roll is its number multiplied by 3.
5. The player's total score is the sum of individual roll scores.

### Input/Output:
- **Input:** A file named `rolls.txt` containing numbers representing the die rolls, one number per line.
- **Output:** A file named `score.txt` containing a single integer representing the final score as per the rules.

### Constraints:
- The input file will contain at least one number, and each number will be between 1 and 6 (inclusive).

### Example:
Suppose the `rolls.txt` file contains:
```
2
5
6
3
4
```
The score will be calculated as:
- 2 (even) -> 2 * 2 = 4
- 5 (odd) -> 5 * 3 = 15
- 6 -> 0
- 3 (odd) -> 3 * 3 = 9
- 4 (even) -> 4 * 2 = 8
Total score: 4 + 15 + 0 + 9 + 8 = 36

The `score.txt` file will contain:
```
36
```

Write a Python program to execute this task.
