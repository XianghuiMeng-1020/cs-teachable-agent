## 截图技巧

### 推荐工具
- Windows: `Win + Shift + S` (截图工具)
- 浏览器: F12开发者工具 + 截图
- 全页面截图: 浏览器扩展(如Full Page Screen Capture)

### 截图注意事项
1. **分辨率**: 使用至少1920x1080的分辨率
2. **隐私**: 不要包含真实的敏感信息
3. **标注**: 可以使用箭头和文字标注关键部分
4. **文件名**: 使用描述性名称,如`01-landing-page.png`

### 截图顺序
建议按照用户使用流程顺序截图:
1. 首页
2. 注册
3. 仪表板
4. 创建TA
5. 教学交互
6. 测试页面
7. 掌握度页面
8. 历史页面

## 自动化截图脚本(可选)

如果需要自动化截图,可以使用Playwright或Selenium:

```javascript
// 示例: 使用Playwright
const { chromium } = require('playwright');

(async () => {
  const browser = await chromium.launch();
  const page = await browser.newPage();
  
  // 首页
  await page.goto('http://localhost:3000');
  await page.screenshot({ path: 'screenshots/01-landing.png' });
  
  // 注册
  await page.fill('input[name="username"]', 'test_user');
  await page.fill('input[name="password"]', 'TestPass123!');
  await page.screenshot({ path: 'screenshots/02-register-form.png' });
  await page.click('button[type="submit"]');
  
  // 仪表板
  await page.waitForURL('**/dashboard');
  await page.screenshot({ path: 'screenshots/03-dashboard.png' });
  
  // ... 更多截图
  
  await browser.close();
})();
```
